# Azure Hybrid Infrastructure Lab

This project simulates a realistic enterprise Azure infrastructure environment built with Terraform.

The goal is to design, deploy, secure, monitor and document a hybrid-ready Azure foundation using Infrastructure as Code principles.


## Scope

The project focuses on infrastructure engineering, not application development or pure DevOps.

Main areas covered:

- Azure Resource Groups
- Virtual Networks and Subnets
- Network Security Groups
- Virtual Machines
- Storage Accounts
- Log Analytics
- Azure Monitor
- Backup
- Security
- Governance
- Terraform modularization
- Git and GitHub workflow
- Enterprise documentation

## Business Scenario

A multinational company is extending its traditional on-premises infrastructure into Azure.

The infrastructure team needs a secure, governed and repeatable Azure foundation that can support internal workloads, monitoring, backup and future hybrid connectivity.
